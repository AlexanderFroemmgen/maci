Configure the Git Integration

To enable the Git integration, you need to store your remote credentials in the appsettings.json file. For Github, go to https://github.com/settings/tokens and generate a Personal Access Token. For other remotes, you may need to put your password in the AccessToken field.

  "Git": {
    "RemoteCredentials": {
      "Username": "your username",
      "AccessToken": "your access token"
    }
  }

Configure the Git Remote for a repository

To enable automatic pushing the repository to a remote, just add the remote with the name origin to the underlying repository. The remote needs to use the https:// protocol as the used library doesn't support SSH remotes.

cd $DataLocation/ExperimentTemplates/$TemplateName
git remote add origin https://github.com/username/reponame
