Python Examples

We do not provide a full Python API so far, butprovide a few Python snippets as API examples. We found these snippets helpful to manage larger projects.

Deleting a bunch of Experiments

import httplib
for i in range(1, 100):
    conn = httplib.HTTPConnection('localhost:63658')
    conn.request('DELETE', '/experiments/' + str(i)) 
    resp = conn.getresponse()
    content = resp.read()
    print "Experiment", i, "passed away..."

Resetting Experiments

import httplib
conn = httplib.HTTPConnection('10.0.254.22:63658')
conn.request('POST', '/experiments/17/instances/148231/reset') 
resp = conn.getresponse()
